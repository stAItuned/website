# My Smart Haircut - Assets

This folder contains media assets for the My Smart Haircut product.

## Files:

- **demo.mp4** (1.5MB) - Video demonstration of the product features
- **demo.gif** (5.1MB) - Animated GIF version of the demo

## Usage:

These files are copied to `/public/assets/products/my-smart-haircut/` for web serving.

In the product definition (index.ts):
- image: '/assets/products/my-smart-haircut/demo.gif'
- coverImage: '/assets/products/my-smart-haircut/demo.gif'
- images: [
    '/assets/products/my-smart-haircut/demo.mp4',
    '/assets/products/my-smart-haircut/demo.gif',
  ]
