# Style Check

This skill ensures that the codebase adheres to the defined style guidelines and passes all unit tests.

## Usage

Run this skill before pushing changes or when verifying a task.

## Steps

1.  **Linting**: Run `npm run lint` to check for code style issues.
2.  **Testing**: Run `npm run test` to execute unit tests.

## Commands

```bash
npm run lint && npm run test
```
