# PWA Screenshots

This folder contains screenshots used in the PWA manifest for the install prompt.

## Required Screenshots

### Desktop (Wide)
- **File**: `screenshot-wide.png`
- **Size**: 1280x720 pixels
- **Format**: PNG
- **Content**: Homepage or articles list on desktop view

### Mobile (Narrow)
- **File**: `screenshot-mobile.png`  
- **Size**: 750x1334 pixels (iPhone standard)
- **Format**: PNG
- **Content**: Articles list on mobile view

1. Open Chrome DevTools â†’ Toggle device toolbar
2. For wide: Set viewport to 1280x720
3. For mobile: Set to iPhone 12 Pro (390x844) then export at 2x
4. Take screenshots of `/learn/articles` page
5. Save as PNG in this folder

## Tips
- Use light theme for better app store visibility
- Show compelling content (articles with images)
- Ensure text is readable at thumbnail size
